# История изменений

В этом файле описаны пользовательские изменения Simple FTP на русском языке.

## [0.2.0] - 2026-06-24

### Добавлено
- Добавлена проверка обновлений при запуске приложения через manifest на GitHub Pages.
- Добавлен диалог обновления в приложении со списком изменений, загрузкой APK, проверкой sha256, проверкой подписи и передачей APK установщику Android.
- Добавлено отображение текущей версии приложения на главном экране.
- Добавлено предупреждение при запуске, если Wi-Fi выключен, с быстрым переходом в настройки Wi-Fi Android.
- Добавлено ограничение клиентских подключений (до 5), чтобы сервер не расходовал ресурсы без ограничений.
- Добавлен таймаут управляющего FTP-сокета 5 минут для неактивных сессий.
- Добавлена явная защита от удаления корневой FTP-папки.
- Добавлена проверка разрешений на чтение и запись на старых версиях Android (API ниже 30).

### Изменено
- Релизные сборки по тегам теперь публикуют manifest обновлений и локализованные changelog-файлы на GitHub Pages.
- Определение FTP-адреса ограничено Wi-Fi-сетями и обновляется после изменений подключения Wi-Fi.
- Оптимизирована генерация QR-кода: размер по умолчанию уменьшен до 256x256, создание bitmap перенесено с главного потока.
- Список папок на экране выбора пути теперь загружается асинхронно в I/O-потоке.
- Обновление выбранного пути сделано потокобезопасным через @Volatile.

## [0.1.0] - 2026-06-22

### Добавлено

- Добавлена пользовательская документация README по установке, использованию, диагностике, сборке и выпуску Simple FTP.
- Добавлены инструкции для Codex-агентов, сохраняющие минимальный FTP-серверный scope для eBookSender и PocketBook.
- Добавлена документация GitHub Actions CI/CD и поддержка release workflow.
